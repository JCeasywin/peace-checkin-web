# MEOI 猫主人视觉人格海报 Skill

## 作品名称
MEOI（Meow Owner Index）猫主人视觉人格海报 Skill

## 作品简介
MEOI 是一个面向养猫人的轻量人格测试产品。用户上传猫照并完成 20 道选择题后，系统会计算 P / C / O / E / X / A 六个维度，判断用户属于 8 种养猫人类型之一，并生成一段可复用的 Wan 2.7 海报生成 Prompt。

作品目标不是做严肃心理测量，而是把“我和猫的关系模式”变成一个准、像、好转发的视觉结果。

## 解决的问题 / 面向场景
养猫用户常常愿意分享猫照，但普通照片缺少新的表达角度。MEOI 把猫照、测试结果、人格标签和 AI 视觉生成结合起来，让用户得到一张更有情绪、更像自己的专属猫主人海报。

适用场景：
- 宠物内容互动活动
- 小红书 / 朋友圈分享
- 宠物品牌拉新互动
- AI 视觉海报生成 Demo
- Wan 2.7 图像生成 / 图像编辑 Skill 示例

## 核心输入
- 用户上传的猫照片
- 20 道 A / B / C 选择题答案
- 系统计算出的养猫人类型
- 类型标签、副标题、正文和底部收尾句

## 核心处理流程
1. 用户上传猫照。
2. 用户完成 20 道测试题。
3. 每题根据映射关系给 P / C / O / E / X / A 六个维度之一加 2 分。
4. 系统按既定判定顺序输出 8 种养猫人类型之一。
5. 页面生成结构化结果卡，包括类型、标签、正文和维度分数。
6. 页面自动拼接 Wan 2.7 Prompt。
7. 用户复制 Prompt 到 Wan 2.7，或后续通过 API 自动调用 Wan 生成视觉海报。
8. 页面展示 Before / After：普通测试结果 vs Wan 视觉海报方向。

## 使用到的 Wan 能力
计划使用 Wan 2.7 图像生成 / 图像编辑能力：
- 图生图：基于用户上传猫照进行视觉化海报生成。
- 文生图：根据养猫人类型、标签和文案生成主题海报。
- 图像编辑：保留猫作为主视觉，生成适合分享的竖版视觉结果。

当前网页版本已完成 Prompt 生成和 Skill 工作流展示；API 自动调用可作为下一版本接入。

## Prompt 模板
```text
请基于用户上传的猫照片，生成一张竖版中文人格测试结果海报。

项目名称：MEOI（Meow Owner Index）
测试主题：你是哪一种养猫人
主人类型：{{result_type}}
副标题：{{subtitle}}
三个标签：{{tags}}
结果正文：{{body}}
底部收尾：{{closing}}

画面要求：
1. 以猫作为主视觉，不出现真人。
2. 画面适合社交媒体分享，竖版 3:4 或 9:16 构图。
3. 风格要有趣、干净、有宠物生活感，文字区域清晰可读。
4. 画面中不要出现未经授权的品牌 Logo、明星脸、影视动漫角色。
5. 标注“AI 生成”，保留 MEOI / Meow Owner Index 字样。

输出目标：
把结构化测试结果视觉化成一张更有情绪、更好转发的 Wan 2.7 AI 海报。
```

## 前后对比
Before：用户只能得到一个普通测试结果，包括类型、标签、正文和分数。

After：用户可以把同一份结果交给 Wan 2.7，生成一张专属视觉海报，让“养猫人格”变成可传播的图片内容。

## 8 种输出类型

- 情绪型共生者

## 合规说明
- 请勿上传包含他人肖像、明星、品牌 Logo 或受版权保护内容的图片。
- 生成结果发布时应标注 AI 生成。
- 当前 Demo 不包含暴力、色情、宗教歧视、政治敏感或违法内容。
- 当前 Demo 默认不使用真人肖像，不复刻第三方 IP 角色。
